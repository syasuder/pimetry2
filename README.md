# pimetry2

UDPでJSON風文字列を受け取ってタコメータ風のウィンドウを表示するQtアプリのサンプルです。

## ビルド

QJsonを利用しています。pimetry2.proと同じ階層にQJsonのコンパイル済みソース一式を配置してビルドしてください。
https://github.com/flavio/qjson

## 環境

* Windows8.1
* Qt SDK 4.8.6
* Qt Creator 3.3.2
* mingw
* QJson (5e3b9b84b225f4612e09ef05aa6d8c23379e4e71)
